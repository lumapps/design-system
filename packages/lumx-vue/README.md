# LumApps Design System (LumX)

The first official LumApps Design System for [Vue][vue] applications.
_LumX_ will help you design your applications faster and more easily. You will be able to use LumX with the LumApps design styleguides.

## Quick start

To start to use LumX, you can either:

-   Install with Yarn/NPM:

```bash
yarn add @lumx/vue
# Or
npm install @lumx/vue
```

-   Download the latest release for [Vue][vue-release]
-   Clone the repository: `git clone https://github.com/lumapps/design-system.git`

## Documentation

You can find an online version of the demo and documentation site at https://design.lumapps.com.
