# Color

**Use color utility classes to add color to fonts and backgrounds.**

Use color utility based on [color palette](/product/foundations/color) variants.

## Font color

```jsx demo
export default () => <span className="lumx-color-font-dark-L2">The quick brown fox jumps over the lazy dog</span>;
```

## Background color

```jsx demo
export default () => (
    <div className="lumx-color-background-primary-N lumx-spacing-padding-big">
        <span className="lumx-color-font-light-N">The quick brown fox jumps over the lazy dog</span>
    </div>
);
```
