# Tooltip <ReactStabilityFlag status="stable" />

**Tooltips display information when users hover or focus an element.**
They are positioned using anchor elements.

<DemoBlock orientation="horizontal" demo="default" />

## Placement

Tooltips are positioned at the bottom of the target by default.
Tooltips' position can be forced to one of these options: `top`, `right`, `bottom`, or `left`.
If there is not enough space to correctly display the tooltip, tooltip will be displayed at the best position.
