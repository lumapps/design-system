# Message

**Displays a prominent and permanent message.**

## Typology

### Info

Informative messages display advices, tips and/or informations related to another component or pattern.

<DemoBlock demo="info" />

### Warning

Warning messages display very important information (often about a limitation) related to another component or pattern.

<DemoBlock demo="warning" />

### Error

Error messages display critical information (often about a loss for the user) related to another component or pattern.

<DemoBlock demo="error" />

### Success

Succes messages display critical information (often about the completion of a task performed by the user) related to another component or pattern.

<DemoBlock demo="success" />
