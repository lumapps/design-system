# Toolbar

**Toolbars display information and actions related to the context.**

```jsx demo
import { mdiMagnify } from '@lumx/icons';
import { IconButton, Toolbar } from '@lumx/react';
import classNames from 'classnames';

export default ({ theme = 'light' }) => (
    <Toolbar
        label={
            <span
                className={classNames(
                    'lumx-typography-title',
                    theme === 'light' ? 'lumx-color-font-dark-N' : 'lumx-color-font-light-N',
                )}
            >
                Toolbar title
            </span>
        }
        after={<IconButton label="Search" emphasis="low" theme={theme} icon={mdiMagnify} />}
    />
);
```

## Anatomy

A toolbar has three sections. An optional contextual icon button and a title on the left, contextual actions on the right.

## Examples

### Dialog toolbar

```jsx demo
import { mdiMenuDown, mdiTranslate } from '@lumx/icons';
import { Button, Toolbar } from '@lumx/react';
import classNames from 'classnames';

export default ({ theme = 'light' }) => (
    <Toolbar
        label={
            <span
                className={classNames(
                    'lumx-typography-title',
                    theme === 'light' ? 'lumx-color-font-dark-N' : 'lumx-color-font-light-N',
                )}
            >
                Toolbar title
            </span>
        }
        after={
            <Button emphasis="low" theme={theme} leftIcon={mdiTranslate} rightIcon={mdiMenuDown}>
                Value
            </Button>
        }
    />
);
```

### Media picker toolbar

```jsx demo
import { mdiMagnify, mdiViewGrid } from '@lumx/icons';
import { Button, FlexBox, IconButton, TextField, Toolbar } from '@lumx/react';
import classNames from 'classnames';
import { useState } from 'react';

export default ({ theme = 'light' }) => {
    const [value, setValue] = useState('');
    return (
        <Toolbar
            label={
                <FlexBox orientation="horizontal" hAlign="center">
                    <span
                        className={classNames(
                            'lumx-typography-title',
                            theme === 'light' ? 'lumx-color-font-dark-N' : 'lumx-color-font-light-N',
                        )}
                    >
                        Toolbar title
                    </span>

                    <TextField
                        className="lumx-spacing-margin-left-big"
                        value={value}
                        onChange={setValue}
                        icon={mdiMagnify}
                        theme={theme}
                    />
                </FlexBox>
            }
            after={
                <FlexBox orientation="horizontal" hAlign="center">
                    <IconButton
                        label="Grid"
                        className="lumx-spacing-margin-right-regular"
                        emphasis="low"
                        theme={theme}
                        icon={mdiViewGrid}
                    />

                    <Button theme={theme}>Action</Button>
                </FlexBox>
            }
        />
    );
};
```

### File list widget

```jsx demo
import { mdiChevronLeft, mdiMagnify, mdiOpenInNew, mdiSort } from '@lumx/icons';
import classNames from 'classnames';
import { useState } from 'react';
import { Button, FlexBox, IconButton, TextField, Toolbar } from '@lumx/react';

export default ({ theme = 'light' }) => {
    const [value, setValue] = useState('');
    return (
        <Toolbar
            before={<IconButton label="Previous" emphasis="low" theme={theme} icon={mdiChevronLeft} />}
            label={
                <FlexBox orientation="horizontal" hAlign="center" gap="regular">
                    <span
                        className={classNames(
                            'lumx-typography-title',
                            theme === 'light' ? 'lumx-color-font-dark-N' : 'lumx-color-font-light-N',
                        )}
                    >
                        Folder name
                    </span>
                    <IconButton label="Previous" emphasis="low" theme={theme} icon={mdiOpenInNew} />
                </FlexBox>
            }
            after={
                <FlexBox orientation="horizontal" hAlign="center" gap="regular">
                    <TextField value={value} onChange={setValue} icon={mdiMagnify} theme={theme} />
                    <IconButton label="Grid" emphasis="low" theme={theme} icon={mdiSort} />
                    <Button theme={theme}>New</Button>
                </FlexBox>
            }
        />
    );
};
```

### Accessibility concerns

Toolbars are simple layout blocks without any particular semantic or ARIA role.
