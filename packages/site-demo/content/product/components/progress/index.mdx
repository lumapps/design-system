# Progress

**Progress notify user that an action is being completed.**\
Progress come in two styles: `circular` and `linear`.

## Circular

```jsx demo orientation="horizontal" withThemeSwitcher
import { ProgressCircular } from '@lumx/react';

export default ({ theme }) => <ProgressCircular theme={theme} />;
```

## Linear

```jsx demo withThemeSwitcher
import { ProgressLinear } from '@lumx/react';

export default ({ theme }) => <ProgressLinear theme={theme} />;
```

### Accessibility concerns

Progress elements are purely visual and do not use any special accessibility or landmark role.

Please look into ARIA live regions to announce the loading state and changes to the user.
