# Skeleton

**Skeleton are primitives shapes which mimic a piece of content in a recognizable way. It communicate [loading states](/product/foundations/loading-state).**

There are three variants: `circle`, `rectangular`, `typography`. When designing loading states with skeletons, use a combination of variants that best mimic the replaced content.

## Circle

Use circle skeletons to mimic components like [avatar](/product/components/avatar/), [icon](/product/components/icon). Respect at best the size of replaced element using the T-shirt [sizes](/product/foundations/sizes).

<DemoBlock demo="circle" orientation="horizontal" withThemeSwitcher />

## Rectangle

Use rectangular skeletons to mimic [thumbnail](/product/components/thumbnail), [button](/product/components/button), [chip](/product/components/chip), [text field](/product/components/text-field) and so on. `width`, `height` and `border-radius` are adjustable.

<DemoBlock demo="rectangle" orientation="horizontal" withThemeSwitcher />

## Typography

Use typography skeletons to mimic text. Use [text style](/product/foundations/typography) as parameter to easily set a replacement for texts. `width`, is adjustable.

<DemoBlock demo="typography" withThemeSwitcher />

## Animation and style

Skeletons have a unique animation and style: pulsating grey or white depending on the theme.
