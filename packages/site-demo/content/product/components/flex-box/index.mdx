# Flex box

**Quickly manage the layout, alignment, and sizing of grid columns, navigation, components.**

If you are new to or unfamiliar with flexbox, we encourage you to read this <a href="https://css-tricks.com/snippets/css/a-guide-to-flexbox/" target="_blank" rel="noreferrer">CSS-Tricks flexbox guide</a>, the component provide transpositions of flexbox CSS properties.

### Accessibility concerns

Flex boxes are simple layout blocks without any particular semantic or ARIA role. They use `<div>` by default, but it can be customized with the `as` property.
