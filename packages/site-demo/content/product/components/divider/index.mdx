# Divider

**Use dividers to structure layouts.**

```jsx demo withThemeSwitcher
import { Divider } from '@lumx/react';

export default ({ theme }) => <Divider theme={theme} className="lumx-spacing-margin-vertical-big" />;
```

### Accessibility concerns

Dividers use standard `hr` HTML elements and thus inherit the [`separator` role](https://developer.mozilla.org/en-US/docs/Web/Accessibility/ARIA/Roles/separator_role).
