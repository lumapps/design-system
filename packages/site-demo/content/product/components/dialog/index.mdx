# Dialog

**Inform users about a task and can contain critical information, require decisions, or involve multiple tasks.**

```jsx demo orientation="horizontal" hasPlayButton
import { mdiPlay } from '@lumx/icons';

    const buttonRef = useRef(null);
    return (
        <>
            <Button leftIcon={mdiPlay} ref={buttonRef} onClick={toggle} theme={theme}>
                Try dialog
```

## Anatomy

Dialogs can have three sections: header ([toolbar component](/product/components/toolbar)), body and footer ([toolbar component](/product/components/toolbar)).
On user scroll, the header stays fixed on top and the footer at the bottom of the dialog.

## Typology

### Confirm dialog

Confirmation dialogs cannot be dismissed until users confirm a choice.

```jsx demo orientation="horizontal" hasPlayButton
import { mdiPlay } from '@lumx/icons';
import { Button, Dialog, Toolbar } from '@lumx/react';
import { useCallback, useRef, useState } from 'react';

export default ({ theme }) => {
    const [isOpen, setOpen] = useState(false);
    const close = useCallback(() => setOpen(false), []);
    const toggle = useCallback(() => setOpen(!isOpen), [isOpen]);
    const buttonRef = useRef(null);
    return (
        <>
            <Button leftIcon={mdiPlay} ref={buttonRef} onClick={toggle} theme={theme}>
                Try confirm dialog
            </Button>

            <Dialog size="tiny" isOpen={isOpen} parentElement={buttonRef} onClose={close} preventAutoClose>
                <header>
                    <Toolbar label={<span className="lumx-typography-title">Confirm dialog</span>} />
                </header>

                <p className="lumx-spacing-padding-horizontal-huge">
                    Lorem ipsum dolor sit amet, consectetur adipiscing elit. Praesent sit amet urna quis nisi sodales
                    semper pharetra eu augue.
                </p>

                <footer>
                    <Toolbar
                        after={
                            <>
                                <Button emphasis="medium" onClick={close}>
                                    Disagree
                                </Button>

                                <Button className="lumx-spacing-margin-left-regular" onClick={close}>
                                    Agree
                                </Button>
                            </>
                        }
                    />
                </footer>
            </Dialog>
        </>
    );
};
```

## Sizes

Dialogs come in four sizes: `tiny` (400dp), `regular` (600dp), `big` (800dp), and `huge` (full screen).

```jsx demo orientation="horizontal" hasPlayButton
import { mdiPlay } from '@lumx/icons';
import { Button, Dialog, Toolbar } from '@lumx/react';
import { useCallback, useRef, useState } from 'react';

export default ({ theme }) => {
    const [size, setSize] = useState();
    const [isOpen, setOpen] = useState(false);
    const close = useCallback(() => setOpen(false), []);
    const toggle = useCallback(() => setOpen(!isOpen), [isOpen]);
    const buttonRef = useRef(null);
    const onClickSize = (newSize) => () => {
        setSize(newSize);
        toggle();
    };
    return (
        <>
            <Button
                className="lumx-spacing-margin-horizontal-tiny"
                leftIcon={mdiPlay}
                ref={buttonRef}
                onClick={onClickSize('tiny')}
                theme={theme}
            >
                Tiny
            </Button>

            <Button
                className="lumx-spacing-margin-horizontal-tiny"
                leftIcon={mdiPlay}
                ref={buttonRef}
                onClick={onClickSize('regular')}
                theme={theme}
            >
                Regular
            </Button>

            <Button
                className="lumx-spacing-margin-horizontal-tiny"
                leftIcon={mdiPlay}
                ref={buttonRef}
                onClick={onClickSize('big')}
                theme={theme}
            >
                Big
            </Button>

            <Button
                className="lumx-spacing-margin-horizontal-tiny"
                leftIcon={mdiPlay}
                ref={buttonRef}
                onClick={onClickSize('huge')}
                theme={theme}
            >
                Huge
            </Button>

            <Dialog isOpen={isOpen} parentElement={buttonRef} onClose={close} size={size}>
                <header>
                    <Toolbar label={<span className="lumx-typography-title">Dialog</span>} />
                </header>

                <p className="lumx-spacing-padding-horizontal-huge">
```
