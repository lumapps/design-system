# Generic block

**A technical block used to build other blocks.**

## Anatomy

Generic blocks have 3 sections : 1) **Figure**, 2) **Content** and 3) **Actions**.

Here presented in two **orientation**: **Vertical** and **Horizontal**.

<DemoBlock orientation="horizontal" vAlign="space-evenly" demo={{ react: DemoAnatomy }} />

### Figure section

Optional, containing a figure-like element (like [Avatar](/product/components/avatar/), [Thumbnail](/product/components/thumbnail/), [Icon](/product/components/icon/), etc.).

<DemoBlock orientation="horizontal" hAlign="bottom" vAlign="space-evenly" demo={{ react: DemoFigureSection }} />

Containing one or more content elements ([Heading](/product/components/heading/), [Text](/product/components/text/), [Icon](/product/components/icon/), [Link](/product/components/link), [InlineList](/product/components/inline-list/), [User block](/product/components/user-block/), etc.).

## Alignment

### Alignments in horizontal orientation

On **Horizontal** orientation, generic blocks can be aligned: `top`, `center` or `bottom`.

<DemoBlock orientation="horizontal" vAlign="space-evenly" demo={{ react: DemoAlignmentsHorizontal }} />

### Alignments in vertical orientation

On **Vertical** orientation, generic blocks can be aligned: `right`, `center` or `left`.

<DemoBlock orientation="horizontal" vAlign="space-evenly" demo={{ react: DemoAlignmentsVertical }} />

### Accessibility concerns

Generic blocks are mostly layout block. They use `div` by default but you can customize that by changing the `as` prop for on the `GenericBlock`, `GenericBlock.Figure`, `GenericBlock.Content` and `GenericBlock.Actions` components.

You can, for example, use `<GenericBlock as="article">` so that generic blocks are considered as independent, self-contained content on a page.
