# Comment block

**Comment blocks display rich text messages as comments or chat messages.**

```jsx demo withThemeSwitcher
import { mdiHeart, mdiReply } from '@lumx/icons';
import { Button, CommentBlock } from '@lumx/react';

export default ({ theme }) => (
    <CommentBlock
        hasActions
        actions={[
            <Button key="button0" theme={theme} emphasis="low" size="s" leftIcon={mdiHeart}>
                24 likes
            </Button>,
            <Button key="button1" theme={theme} emphasis="low" size="s" leftIcon={mdiReply}>
                Reply
            </Button>,
        ]}
        theme={theme}
        avatarProps={{ image: '/demo-assets/persona.png', alt: 'Avatar' }}
        date="4 hours ago"
        name="Emmitt O. Lum"
        text="All the rumors have finally died down and many skeptics have tightened their lips, the iPod does support video format now on its fifth generation."
    />
);
```

## Relevant

Use relevant comment block for validated messages as an accepted answer to a question.


### Accessibility concerns

Comment blocks and it's default subcomponents do not use any special landmark or accessibility role.

Avoid using the deprecated `onClick`, `onMouseEnter` & `onMouseLeave` props as they do not provide accessible interactive elements.
