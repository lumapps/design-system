# Thumbnail mosaic

**Thumbnail mosaics display multiple formatted images in a constant space.**

## Images number

Thumbnail mosaics can display up to 4 images.

## Click on thumbnail

Thumbnails are clickable. On click, it should open a [slideshow](/product/components/slideshow) in a [lightbox](/product/components/lightbox) with every image.

### Accessibility concerns

See [thumbnail accessibility concerns](/product/components/thumbnail/#accessibility-concerns). Consider adding an explicit label on clickable thumbnails to make sure users know what action they do (open a lightbox, download image, etc.).
