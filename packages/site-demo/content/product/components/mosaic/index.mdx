# Thumbnail mosaic

**Thumbnail mosaics display multiple formatted images in a constant space.**

## Images number

Thumbnail mosaics can display up to 4 images.

### More than four images

If there are more than 4 images to display, thumbnail mosaics can display the resting number on top of the fourth thumbnail.

In this example there are 6 images in total, 4 are shown, 2 aren't.

<DemoBlock demo={{ react: ReactDemoMoreThanFourImages, vue: VueDemoMoreThanFourImages }} />

## Click on thumbnail

Mosaic's thumbnails can be clickable to attach an [image lightbox](/product/components/image-lightbox) that displays the images in a full screen image slideshow.

<DemoBlock demo={{ react: ReactDemoClickOnThumbnail, vue: VueDemoClickOnThumbnail }} />

### Accessibility concerns

See [thumbnail accessibility concerns](/product/components/thumbnail/#accessibility-concerns). Consider adding an explicit label on clickable thumbnails to make sure users know what action they do (open a lightbox, download image, etc.).
