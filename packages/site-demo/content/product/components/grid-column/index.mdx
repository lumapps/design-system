# Grid column

**Display children in a customised grid, with a responsive number of column based on children minimum width constraint.**

## Example

<DemoBlock
    marginAuto={['left', 'right']}
    style={{ overflow: 'hidden', resize: 'horizontal', maxWidth: '100%' }}
    demo={{ react: ReactDemo1, vue: VueDemo1 }}
/>

### Accessibility concerns

GridColumn is a simple layout blocks without any particular semantic or ARIA role. It use `<div>` by default, but it can be customized with the `as` property.
