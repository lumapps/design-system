# Image block

**Image blocks display an image with related info.**
Image blocks support thumbnail's formats (see [thumbnail](/product/components/thumbnail)).

## Custom padding

## Info above

Info can be placed above the image.

<DemoBlock withThemeSwitcher wrap={false} demo={{ react: Demo3 }} />

## Info alignment

There are three alignments: `left`, `center` and `right`.

<DemoBlock withThemeSwitcher wrap={false} demo={{ react: Demo4 }} />

### Accessibility concerns

Image blocks use standard HTML figure element with the `title`, `description` and `tags` displayed in the figure caption.

See [thumbnail accessibility concerns](/product/components/thumbnail/#accessibility-concerns) for the image.
