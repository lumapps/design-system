# Tooltip

**Tooltips display information when users hover over or focus on an element. They are positioned from this element which act as an anchor.**

## Demo

```javascript jsx withThemeSwitcher
() => (
    <>
        Un boutton m qui declenche une tooltip default bottom
    </>
);
```

## Placement

Tooltips can be positioned _top_, _right_, _bottom_, or _left_ to the trigger element.

```javascript jsx withThemeSwitcher
() => (
    <>
        <img width="420" src="demo/react/doc/product/components/tooltip-postion.png" />
    </>
);
```
